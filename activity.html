🏆 กิจกรรม

📚 กิจกรรมด้านวิชาการ

  • เข้าร่วมการแข่งขันตอบปัญหาวิชาการ
  • เข้าค่ายวิชาการ
  • อบรม HTML / CSS และเทคโนโลยี
  • นำเสนอผลงานโครงงาน

❤️ กิจกรรมจิตอาสา

  • เก็บขยะในชุมชน
  • ปลูกต้นไม้
  • ช่วยงานวันเด็ก
  • ช่วยงานวัดและชุมชน

🏫 กิจกรรมในโรงเรียน

  • ช่วยจัดงานกีฬาสี
  • ช่วยจัดงานวันวิทยาศาสตร์
  • ช่วยงานวันไหว้ครู
  • เป็นพิธีกรในกิจกรรมโรงเรียน

💻 ชมรมและผลงาน

  • สมาชิกชมรมคอมพิวเตอร์
  • สร้างเว็บไซต์ Portfolio
  • ศึกษาและพัฒนา HTML / CSS
  • ออกแบบเว็บไซต์ด้วย GitHub Pages

🏅 การแข่งขันและการอบรม

  • แข่งขันสร้างเว็บไซต์
  • อบรม AI
  • อบรม Cyber Security
  • อบรม Python เบื้องต้น
